# AmalgaMate Hub Component

An interactive three-panel brand hub built with React and Vite.

## Project Structure

```
amalgamate-hub/
├── src/
│   ├── components/
│   │   └── AmalgamateHub.jsx      # Main component
│   ├── App.jsx                     # App entry point
│   ├── main.jsx                    # React DOM mount
│   └── index.css                   # Global styles
├── public/                         # Static assets
├── index.html                      # HTML entry point
├── vite.config.js                  # Vite configuration
├── package.json                    # Dependencies & scripts
└── .gitignore                      # Git ignore rules
```

## Installation

1. Clone or download this project
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

Create a production build:
```bash
npm run build
```

## Preview

Preview the production build locally:
```bash
npm run preview
```

## Features

- Hover-to-expand animated panels
- Three brand sections: AmalgaMate, Adventure, Gear
- Smooth CSS transitions and glow effects
- Google Fonts (Bebas Neue, Syne, DM Sans)
- Built with Vite for fast development and optimal builds

## Tech Stack

- **React 18** - UI library
- **Vite 4** - Build tool and development server
- **CSS3** - Styling with animations and gradients
